# cube-timer-app

ルービックキューブのソルブタイムを計測するためのタイマーアプリです。計測のたびにランダムなスクランブルが自動生成され、タイムの記録とともにデータベースで一元管理されます。

## 環境構築

- dockerを使ってmongoDBを起動する

```bash
$ docker run --rm --name=cube-timer-app -p 27017:27017 mongo
```

- serverを起動する

```bash
$ npm install
$ node index.js
```
- ブラウザで http://localhost:3000 にアクセス

## 使い方
1. **インスペクションの開始**: 待機状態でスペースキーを1回押します（15秒のカウントダウンが始まります）。

2. **スタート準備**: インスペクション中にスペースキーを押しっぱなしにします（タイマーの文字が緑色になります）。

3. **計測スタート**: スペースキーから指を離した瞬間に計測が始まります。

4. **計測ストップ**: 計測中に任意のキーを押すとストップし、自動的にデータベースに記録が保存されます。

5. **記録の削除**: 履歴リストの記録（カード）をクリックすると、確認ダイアログが表示され、OKを押すと削除されます。

## 機能
- 15秒のインスペクションタイム (残り5秒になると赤文字で警告)
- スクランブルをソルブごとに自動生成
- ベストタイムを表示
- 過去の記録とスクランブルを参照可能
